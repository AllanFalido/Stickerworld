### Install
Clone this project

```bash
> git clone https://github.com/kaiserdj/Stickerworld.git
> cd Stickerworld
```

Install the dependencies:

```bash
> npm install
```

### Usage
1. run the Whatsapp bot

```bash
> npm start
```
